# AutoHire — Fullstack Starter App

This repository contains a starter implementation of **AutoHire**: a combined freelancer + hiring marketplace.

## Structure
- `backend/` — Node.js + Express + MongoDB API
- `frontend/` — React + Vite frontend

## Quick start

### Backend
```bash
cd autohire/backend
cp .env.example .env
# edit .env to set MONGO_URI and JWT_SECRET
npm install
npm run dev
